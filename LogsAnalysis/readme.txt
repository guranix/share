Пример команды для запуска с параметрами: 
java -jar LogsAnalysis.jar guran -u=guran -h=00 2010-01-01 12:00:12    ^[a-zA-Z]+[a-zA-Z0-9._-]*$ 20 /home/guran/pr1/tut -h=23

Параметры могут идти в любом порядке. параметры разделяются пробелами.

Параметры для фильтрации(хотябы один должен быть указан):
Формат даты: yyyy-MM-dd (Например: 2014-12-29)
Формат времени: HH:mm:ss (Например: 23:59:59)
Даты и время можно указывать в любом порядке, одн или два раза.
Имя пользователя должно начинаться с буквы и может состоять из любых латинских букв и цифр.
Паттерн для строки должен начинаться с символа ^ и заканчиваться символом $ (Например:  ^[a-zA-Z]+[a-zA-Z0-9._-]*$ )

Параметры для группировки (хотябы один должен быть указан):
Для группировки по имени пользователя укажите -u=имя_пользователя (Например: -u=guran)
Для группировки по временному интервалу в 1 указанный час, укажите -h=цифру_от_0_до_23 (Например: -h=23, программа посчитает количество отфильтрованных строк, которые были внесены в логи с 23 до 00 часов)

Другие параметры (не обязательные):
Для задания количества потоков просто укажите целое число большее  0 (Например: 4). Количество потоков по умолчанию = 1;
Для задания пути к выходному файлу укажите полный путь к файлу или к папке (Например /home/username/out.txt или /home/username/). Файл по умолчанию создается в дирректории приложения. По умолчанию файл называется output.txt.
